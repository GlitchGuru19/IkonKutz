# Ikon Cuts

Premium barber booking web application.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Setup Firebase**
   - Follow instructions in `authentication.md` to get your API keys.
   - Update `src/firebase.ts`.

3. **Run Locally**
   ```bash
   npm run dev
   ```

## Features

- **For Customers**: Browse services, check prices, view real-time availability (Sat/Sun), and book appointments.
- **For Admin**: Manage service list, generate and edit schedule availability, view bookings.

## Customization

- Change App Name in `src/config.ts` (Comment added).
- Adjust styles in `src/index.css`.
